↓読んで欲しいところ(環境起動方法等)
--------------------------------------------------------
概要
	・にじさんじライバーがYoutubeでライブを始めると自動で配信画面を開いてくれます。
	・開始してから開くまでに約0～3分ほどかかってしまうので補助ツール程度のレベルで考えてください。
環境
	・OS:ライブ配信が見れるWindowsPC(MacOSは持っていないため対応できない)
	・ブラウザ：Chrome(現在Chromeのみ、その他ブラウザも対応予定)
インストール方法
	・解凍したフォルダをそのまま好きなところに入れてください
	　(「C:\Program Files」の中に入れるのがいいんじゃないの？)
	
手動起動方法
	①にじうぉっち.exeをクリック
	②にじさんじ勢のライブ配信が始まると勝手にChromeからライブ画面が開かれます。
自動起動方法(パソコン起動したら勝手に起動)(Windows10の場合)(7,8でも多分同様に設定できます)
	①Windowsキー + Rボタンを押下
	②ファイル名を指定して実行に「shell:startup」と入力しOKボタンを押下
	③開かれるフォルダーの中に「にじうぉっちゃー.exe」のショートカットを入れる
	④Windowsが起動すると自動でにじうぉっちゃーが起動する
	※その他OSは調べればすぐに出てくると思います。(大体この方法で大丈夫？)
配信ライバー選択
	①タスクトレイ(画面の右下のところ)にあるにじさんじうぉっちを右クリック
	②チェックがあるメンバー？グループ？が自動で起動します。
終了方法
	①タスクトレイ(画面の右下のところ)にあるにじさんじうぉっちを右クリック
	②終了をクリック
	③終
	
起動できないとかやり方がわからないとかがあればお気軽にTwitterに質問ください。
